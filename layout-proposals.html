Advantages

  • Minimal code change (~50 lines)
  • Familiar pattern — toolbar already has controls
  • No layout restructuring needed
  • Works on any screen size
  • Each view gets full chart area
  • Also works in "Flow + Candles" tab (sub-tabs for the flow half)

Tradeoffs

  • Can only see ONE view at a time
  • Switching between bubbles and footprint loses context
  • Pro traders often want momentum + detail simultaneously

  • How pros actually trade (Bookmap + footprint side-by-side)
