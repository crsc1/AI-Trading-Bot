5. GEX/DEX FORMULAS (Lines 90-218 in gex_engine.py)
   ✓ GEX = OI × Γ × S² × 0.01 × 100
     * Gamma is per dollar of underlying
     * S² accounts for convexity
     * 0.01 converts to standardized unit
     * 100 is contract multiplier
   ✓ Call GEX: positive (dealers SHORT calls)
   ✓ Put GEX: negative (dealers SHORT puts, sign flip at line 164)
   ✓ DEX = OI × Δ × 100
   ✓ Call delta: positive, dealers hedge by BUYING stock
   ✓ Put delta: negative, dealers hedge by SELLING stock (but sign flipped for short put exposure)

7. MARKET LEVELS (Lines 99-246 in market_levels.py)
   
   VWAP (lines 151-175):
   ✓ cum_tp_vol = Σ(TP × V) where TP = (H+L+C)/3
   ✓ VWAP = cum_tp_vol / cum_vol
   ✓ Variance = E[TP²] - E[TP]² (correct formula)
   ✓ Std dev = sqrt(variance)
   ✓ Bands: ±1σ, ±2σ, ±3σ
   
   Pivot Points (lines 130-138):
   ✓ Pivot = (H + L + C) / 3
   ✓ R1 = 2 × Pivot - L
   ✓ S1 = 2 × Pivot - H
   ✓ R2 = Pivot + (H - L)
   ✓ S2 = Pivot - (H - L)
   ✓ R3 = H + 2 × (Pivot - L)
   ✓ S3 = L - 2 × (H - Pivot)
   All standard formulas ✓
   
   ATR (lines 219-230):
   ✓ TR = max(H - L, |H - PC|, |L - PC|)
   ✓ ATR = SMA(TR, 14)
   ✓ 5-min ATR ≈ 1-min ATR × 2.236 (√5 scaling, correct)
